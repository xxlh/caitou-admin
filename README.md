# Caitou Admin

https://caitou.ieexx.com

This project is initialized with [Ant Design Pro](https://pro.ant.design). Follow is the quick guide for how to use.

## Environment Prepare

```bash
npm install
```

or

```bash
yarn
```

### Start project

Start project with mock data to develop with backend delayed
```bash
npm start
```

Start project with dev server to develop with backend API
```bash
npm run dev
```

Start project with production server to verify with backend API
```bash
npm run start:pre
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
